# minesweepe-rs
Minesweeper in Rust

## Made With
* [Rust](https://github.com/rust-lang/rust)
* [Piston](https://github.com/pistondevelopers/piston)
* GLFW 3
* OpenGL

## Controls
Button                 | Action
---------------------- | ------------
<kbd>Left Click</kbd>  | Uncover tile
<kbd>Right Click</kbd> | Mark tile as suspected mine
<kbd>N</kbd>           | New game
<kbd>Esc</kbd>         | Quit

## Tile colors
Color  | Meaning
------ | -----------
Grey   | Unclicked
Yellow | Marked tile
Red    | Mine
White  | Safe tile
